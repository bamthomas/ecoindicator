# ecoindicator

Eco indicator for a website based on [ecoindex.fr](http://www.ecoindex.fr) : give dom elements/nb HTTP requests/downloaded size

# build

    # install dependencies
    npm install
    
    # testing
    npm test
    
    # build for production with minification
    npm run build
    
    # jslint 
    npm run lint
    
    # fixing jslint
    npm run lint:fix
